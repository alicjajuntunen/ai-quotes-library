# AI Quotes Library

A curated collection of notable quotes about AI — and the surrounding conversation about
design, craft, and how we build with it — pulled verbatim from talks, podcasts, and articles,
each linked back to its original source.

**→ Browse the live site: https://alicjajuntunen.github.io/ai-quotes-library/**

## What's inside

- **Quotes** — a hand-curated list of extracted lines, each attributed to the person who said
  or wrote it and linked to where it came from.
- **Sources** — the talks, transcripts, and articles the quotes are drawn from, grouped by
  publication.

The library is maintained as an [Obsidian](https://obsidian.md) vault, and a small build step
publishes it as a searchable website via GitHub Pages on every push to `main`.
